# TrackWise | Smart Expense Tracker

**TrackWise** is a premium, high-fidelity financial management web application designed to help you track your wealth with precision and style. Built with a focus on modern aesthetics and data-driven insights, it provides a comprehensive overview of your financial health.

## 🌟 Key Features

-   **Intelligent Dashboard:** A clean, structural interface providing real-time updates on your balance, income, and expenses.
-   **Data Visualization (Charts):** Integrated **Chart.js** support with two powerful views:
    -   **Spending Analysis:** A doughnut chart breaking down expenses by category (Grocery, Clothes, Food, etc.).
    -   **Monthly Trends:** A bar chart visualizing your income vs. expenses month-over-month.
-   **Category-wise Tracking:** Organize your spending with predefined categories for better financial oversight.
-   **Recurring Reminders:** Mark transactions as recurring and track your monthly commitments in a dedicated panel.
-   **Financial Health Indicators:** Visual progress bars for an instant comparison of earnings versus spending.
-   **Motivation Hub:** Dynamic financial wisdom and motivation quotes generated every time you visit.
-   **Data Persistence:** Your transactions are securely stored in your browser's Local Storage.
-   **Premium Design:** A "hand-crafted" forced-light theme with modern typography (Outfit) and sophisticated layout.

## 🚀 Getting Started

1.  Clone or download this repository.
2.  Open `expensetracker.html` in any modern web browser (Chrome, Safari, Firefox).
3.  Start recording your transactions!

## 🛠️ Built With

-   **HTML5:** Semantic structure for a robust web experience.
-   **CSS3:** Custom styles with a "Studio" aesthetic and responsive grid layout.
-   **JavaScript (ES6+):** Dynamic logic for state management, data filtering, and local storage.
-   **Chart.js:** Industry-standard library for professional-grade data visualization.
